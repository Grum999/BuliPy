Class description - View

Build from View.h
＋Krita 4.0.0↻Krita 5.2.0

foreGroundColor() ➞ ManagedColor
＋Krita 4.0.0

allows access to the currently active color. This is nominally per canvas/view, but in practice per mainwindow.

color = Application.activeWindow().activeView().foreGroundColor()

components = color.components()

components[0] = 1.0

components[1] = 0.6

components[2] = 0.7

color.setComponents(components)
